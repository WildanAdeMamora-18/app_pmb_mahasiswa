# 📱 Aplikasi Penerimaan Mahasiswa Baru (PMB)

Aplikasi mobile berbasis Flutter yang dirancang untuk mempermudah proses **pendaftaran mahasiswa baru** secara digital. Aplikasi ini menyediakan form input data lengkap, navigasi yang sederhana, serta tampilan UI yang modern dan user-friendly.

---

## 🚀 Fitur Utama

* 📝 Form pendaftaran mahasiswa baru
* 👤 Input data pribadi (Nama, NIK, Email, dll)
* 📅 Input tempat & tanggal lahir (Date Picker)
* 🎓 Pilihan program studi (Dropdown)
* ✔️ Validasi input form
* 🏠 Halaman beranda (Home Screen)
* 🔄 Navigasi antar halaman
* 📱 Tampilan UI modern dan responsif

---

## 🛠️ Teknologi yang Digunakan

* **Flutter**
* **Dart**
* **Material Design**

## ⚙️ Cara Menjalankan Project

1. Clone repository:

```bash
git clone https://github.com/username/pmb-flutter-app.git
```

2. Masuk ke folder project:

```bash
cd pmb-flutter-app
```

3. Install dependencies:

```bash
flutter pub get
```

4. Jalankan aplikasi:

```bash
flutter run
```

---

## 📦 Build APK

Untuk membuat file APK:

```bash
flutter build apk
```

File APK akan tersedia di:

```
build/app/outputs/flutter-apk/app-release.apk
```
